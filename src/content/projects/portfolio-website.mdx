# Developer Portfolio Website

A showcase of modern web development practices using Astro's latest features.

## Overview

This portfolio demonstrates the power of Astro 5.0 for building fast, content-focused websites. It leverages static site generation, content collections, and selective hydration to achieve excellent performance scores.

## Key Features

- **Content Collections**: Type-safe blog and project management
- **View Transitions**: Smooth navigation between pages
- **Performance Optimized**: Lighthouse score > 90
- **Fully Accessible**: WCAG AA compliant

## Technical Highlights

The site uses Astro's island architecture to minimize JavaScript:

```typescript
// Only interactive components are hydrated
<HeroSection client:load />
<MobileMenu client:visible />
```

## Challenges & Solutions

One challenge was optimizing images while maintaining quality. We solved this by:
1. Converting all images to WebP format
2. Implementing lazy loading
3. Using Astro's Image component for automatic optimization

## Results

- Performance Score: 95+
- SEO Score: 100
- Accessibility Score: 100
- Zero layout shift (CLS: 0)
