# Getting Started with Astro 5.0

Astro 5.0 brings powerful new features that make building fast, content-focused websites easier than ever. In this guide, we'll explore the key features and how to get started.

## Why Astro?

Astro is designed for building content-rich websites with optimal performance. It ships zero JavaScript by default, only hydrating interactive components when needed.

```javascript
// Example: Creating a simple component
export default function Welcome({ name }) {
  return <h1>Welcome, {name}!</h1>;
}
```

## Key Features

- **Content Collections**: Type-safe content management with Zod schemas
- **View Transitions**: Smooth page transitions without full reloads
- **Island Architecture**: Selective hydration for optimal performance

## Getting Started

Install Astro using your preferred package manager:

```bash
npm create astro@latest
```

That's it! You're ready to build amazing websites with Astro.
