# Web Performance Optimization Techniques

Performance is crucial for user experience and SEO. Let's explore practical optimization strategies.

## Image Optimization

Images are often the largest assets on a page. Optimize them by:

1. Using modern formats like WebP
2. Implementing lazy loading
3. Serving responsive images

```html
<img 
  src="image.webp" 
  loading="lazy" 
  alt="Description"
/>
```

## Code Splitting

Split your JavaScript bundles to load only what's needed:

```javascript
// Dynamic imports
const module = await import('./heavy-module.js');
```

## Minimize JavaScript

Ship less JavaScript by:
- Using static site generation
- Selective hydration
- Tree shaking unused code

## Measuring Performance

Use tools like Lighthouse to measure:
- First Contentful Paint (FCP)
- Largest Contentful Paint (LCP)
- Time to Interactive (TTI)

Aim for a Lighthouse score above 90 for optimal performance.
